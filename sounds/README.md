# Sounds / SFX

Place your sound effect files here:

- `rain.mp3` — Rain/thunder ambiance
- `steps.mp3` — Footsteps
- `open_door.mp3` — Door opening/closing

These are used as optional background SFX that match the story content.
You can find free SFX at [freesound.org](https://freesound.org) or [pixabay.com/sound-effects](https://pixabay.com/sound-effects/).
